Landing для проекта
Суть: ИИ-ассистент (на базе LLM), который читает финансовые новости и превращает их в понятные торговые сигналы. Пользователь подключает бота по API к своему брокеру (например, Альфа-Банк). Бот не торгует сам "вслепую" (режим Copilot) — он присылает новость, объясняет почему акция вырастет, и дает кнопку "Одобрить сделку".
Главная фича безопасности: Полный запрет на маржинальную торговлю (торговлю в долг/с плечом) на уровне кода. Пользователь не может уйти в минус. Доступ по API только на чтение и торговлю (без вывода средств).
Монетизация: Подписка (для подсказок) ИЛИ Процент с дохода (Success Fee) — мы зарабатываем только тогда, когда клиент в плюсе.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
